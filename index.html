Melissa B. Derecola

Welcome to my portfolio page created for the Creating History in New Media class at George Mason University. This class is part of the Masters in Art History curriculum and is designed to explore how historical information can be presented in a digital environment that is both informative and inviting to others. This reaches beyond basic HTML understanding to exploring site organization and the production of a user friendly interface. From this portfolio page you are invited to explore my course projects including; a responsive design page which demonstrates understanding of aesthetic principles for communication while adapting to a range potential end user page widths, my design rational for my final project including the selected color scheme and type I will use, and finally my final project a separate site using the principles explored in the proceeding projects.
